Qué ocurre cuando una página carga un anuncio

Esta página sirve para explicar a los alumnos la anatomía de una impresión publicitaria.

Cuando el usuario abre una página, el navegador carga el contenido editorial y también las etiquetas publicitarias. Estas etiquetas hacen una llamada al ad server para solicitar un anuncio compatible con esa posición.

El ad server decide qué campaña debe servirse según prioridad, segmentación, disponibilidad de inventario, frecuencia, precio y reglas comerciales.

En una operación real, si no hay una campaña directa disponible, el inventario puede pasar a demanda programática: AdSense, Ad Exchange, SSPs, Open Bidding o Header Bidding.

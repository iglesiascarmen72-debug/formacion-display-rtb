# MediaLab

Mini web estática para formación de Display & RTB.

## Páginas incluidas

- `index.html`: home del publisher.
- `articulo.html`: página editorial con formatos in-read y sidebar.
- `video.html`: ejemplo de página con vídeo.
- `formatos.html`: catálogo visual de formatos publicitarios.

## Formatos demo

- Leaderboard 728x90
- MPU / Robapáginas 300x250
- Half Page 300x600
- Billboard 970x250
- InRead 300x250
- Outstream Video
- Sticky Ad

## Cómo abrirla

Abre `index.html` en el navegador.

También puedes publicarla gratis en:

- GitHub Pages
- Netlify
- Cloudflare Pages

## Siguiente paso

Cuando tengas Google Ad Manager, sustituye los bloques `.ad-slot` por etiquetas GPT reales.
